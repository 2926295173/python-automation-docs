## 模拟器
对比真机而言, 模拟器通常可以做到更多。这里推荐使用 `逍遥/MuMu` 模拟器。
```shell
夜神模拟器 adb connect 127.0.0.1:62001
逍遥模拟器 adb connect 127.0.0.1:21503
mumu模拟器 adb connect 127.0.0.1:7555
```
```shell
adb connect 127.0.0.1 # 默认连接本地 5555 端口
```
